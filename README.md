# ProofedQuest

Prove your skills. Hunt proof badges on Solana.

## Live Demo
https://v0-proofedquest-landing-page.vercel.app/

## Smart Contract
- Network: Solana Devnet
- Address: `YOUR_CONTRACT_ADDRESS_HERE`

## Features
- Complete daily quests
- GenLayer AI validation
- Mint proof badges as NFTs
- Trade on Solana marketplace
- Guild recruitment system

## Built With
- V0 (Frontend UI)
- Noah AI (Smart Contracts)
- Solana (Blockchain)
- GenLayer (AI Validation)

## How It Works
1. Connect Solana wallet
2. Complete skill quests
3. GenLayer validates with AI confidence scores
4. Mint proof badge NFT on Solana
5. Trade or get recruited by guilds
